# PulseAtlas

PulseAtlas is a self-contained front-end prototype for a local-first maps app focused on things to do nearby and the social context around them.

It combines:

- map discovery with live pins and route context
- ephemeral stories and friend presence
- review and thread-style venue context
- live friction reports for movement and crowding

## Run

Open `index.html` directly in a browser, or serve the folder with a static file server.

## Included interactions

- search by vibe, spot name, or tag
- filter by category and intent
- switch between `Discover`, `Move`, and `Social` modes
- inspect map pins and venue cards
- toggle `Ghost mode` for privacy-aware social presence
- rebuild a suggested night-out itinerary
